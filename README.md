# toothcare-comparison

Compare tooth care insurance proposals

# Installation

0. Install node, npm and git
1. Open command prompt (bash, cmd etc.)
1. git clone https://github.com/r4nd0m/toothcare-comparison
2. cd toothcare-comparison
3. npm install
4. ng serve
